How to Draw Line :

Cara Menggambar Polygon :

  1. Klik tombol "POLYGON"
  2. Masukkan jumlah titik (default terdapat 3 titik)
  3. Klik canvas sebanyak jumlah titik yang sudah dimasukkan

Cara Mengubah Warna Polygon :

  1. Klik tombol "COLOR"
  2. Pilih warna yang diinginkan menggunakan color input
  3. Klik polygon yang ingin diubah warnanya

Cara Meng-save Hasil Pekerjaan :

  1. Klik tombol "SAVE"
  2. Pilih folder tempat file akan disimpan

Cara Meng-load Hasil Pekerjaan :

  1. Klik tombol input file
  2. Klik tombol "LOAD"
